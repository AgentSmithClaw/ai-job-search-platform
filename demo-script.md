# GapPilot Platform — 60 秒演示脚本

## 演示目标
用 60 秒展示 GapPilot 的核心价值：**上传简历 + 粘贴 JD → AI 分析差距 → 给出行动建议**。

---

## 演示流程

### 第 1 步：注册登录（5 秒）
- 打开平台首页
- 输入邮箱，点击注册
- 自动登录，进入仪表盘

### 第 2 步：查看首页仪表盘（5 秒）
- 展示：分析次数、投递数、任务数、面试数
- 说明：这是一个完整的求职管理系统，不只是分析工具

### 第 3 步：粘贴 JD（10 秒）
- 点击"开始分析"
- 粘贴一份目标岗位 JD（如：AI 研究助理）
- 展示 JD 输入区域

### 第 4 步：上传简历（10 秒）
- 上传一份 PDF 简历
- 展示 OCR 解析进度
- 简历内容被自动提取

### 第 5 步：AI 分析（10 秒）
- 点击"开始分析"
- 展示加载动画（skeleton screen）
- 等待分析完成

### 第 6 步：查看分析报告（15 秒）
- **匹配度评分**：整体匹配度百分比
- **优势项**：简历中与 JD 匹配的能力
- **差距项**：需要补充的能力，每项可一键复制
- **定制简历建议**：AI 给出的简历改写方向
- **学习任务建议**：从差距项自动生成待学习任务

### 第 7 步：导出报告（5 秒）
- 点击"导出 DOCX"
- 下载结构化分析报告
- 说明：可以直接发给团队或存档

---

## 关键台词

> "GapPilot 解决的问题是：求职时最花时间的不是投简历，而是判断'这个岗位我该不该投'和'简历该怎么改'。上传简历、粘贴 JD，AI 帮你自动分析差距、生成改写建议、规划学习路径。"

## 演示数据

### 示例 JD
```
AI 研究助理（广州 4399）
职责：研究 AI 工具与方法在业务流程中的应用场景，形成结构化方案与试点建议。
要求：熟悉大模型应用形态（Prompt / RAG / Agent），有流程分析和文档输出能力。
```

### 示例简历要点
- 5 年需求分析与项目推进经验
- 有 RPA 流程自动化实施经验
- 做过 AI Meeting Copilot 原型
- 擅长结构化文档输出

### 预期分析结果
- 匹配度：约 72%
- 优势：流程分析、项目推进、结构化输出
- 差距：大模型底层原理、论文跟踪经验
- 建议：补充 RAG 实践案例，优化简历中 AI 相关项目描述

## 技术亮点（面试可讲）

1. **简历解析多层降级**：PDF 文字提取 → OCR 扫描件兜底
2. **LLM 模型路由**：支持 OpenAI / MiniMax 等多模型切换
3. **Mock + Stripe 双模式支付**：未配置 API Key 也能完整体验
4. **本地可信度校验**：LLM 输出二次校验，减少幻觉
5. **49 个 pytest 测试**：后端核心逻辑全覆盖
